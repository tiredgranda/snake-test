# 🐍 Great Place To Work — Snake Game

A local snake game built in Python using `pygame`, styled in Great Place To Work brand colors.

## Requirements

- Python 3.8+
- pygame

## Installation

```bash
git clone https://gitlab.com/your-org/snake-gptw.git
cd snake-gptw
pip install pygame
python snake.py
```

## Controls

| Key | Action |
|---|---|
| Arrow Keys | Move snake |
| SPACE | Start / Restart |
| ESC | Return to menu |

## Colors

| Element | Hex |
|---|---|
| Background | `#11131C` |
| Snake head | `#FF1628` |
| Snake body | `#69737A` |
| Food | `#FF1628` |
| HUD panel | `#32383F` |
